## Запуск

Для запуска на локальном рабочем месте используется дев-сервер, поднимающийся по адресу:

```
    http://localhost:5173
```

Команда для запуска:

```
npm run dev
```

### Запуск ESLint

```
npm run lint:js
```

### Запуск Stylelint

```
npm run lint:css
```

### Запуск Prettier

```
npm run prettier
```

### Сборка

```
npm run build
```
